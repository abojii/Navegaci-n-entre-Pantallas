# Navegación con Jetpack Compose

Aplicación básica desarrollada en Android Studio utilizando Kotlin y Jetpack Compose.

## Objetivo

Implementar navegación entre múltiples pantallas usando Navigation Compose.

## Pantallas incluidas

1. Pantalla de Inicio
2. Pantalla de Detalle
3. Pantalla de Perfil

## Funcionalidades

- Navegación funcional entre pantallas.
- Uso de botones interactivos.
- Implementación de NavController, NavHost y composable routes.
- Flujo continuo entre Inicio, Detalle y Perfil.

## Tecnologías utilizadas

- Kotlin
- Jetpack Compose
- Navigation Compose
- Android Studio

## Cómo ejecutar el proyecto

1. Clonar el repositorio.
2. Abrir el proyecto en Android Studio.
3. Esperar la sincronización de Gradle.
4. Ejecutar la app en un emulador o dispositivo Android.
